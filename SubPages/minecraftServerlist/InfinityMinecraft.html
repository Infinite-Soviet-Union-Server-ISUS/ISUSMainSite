Infinity Server for Vanilla Minecraft

This Minecraft server is vanilla with a few plugins, some of which allow bedrock players to join. That means anyone can play together from PC, Mac, Linux, Android, iOS, Xbox, Switch, and PlayStation.

IP Address: sovinity.com


Instructions to join:

In Java Edition, simply add a server and join like normal.
On Windows 10 and mobile devices, add a server with 25565 as the port.
On console editions (Switch, Xbox, Playstation) you will need to change your primary DNS server to 104.238.130.180 to connect to the server.
