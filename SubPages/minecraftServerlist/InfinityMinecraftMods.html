Modded Minecraft - Infinity Server

This server is running All The Mods 6. To join, you will need to install version 1.7.9 of ATM6 on Curseforge. Don't forget to allocate at least 5GB of ram (I recommend about 8) in the Curseforge settings, otherwise it will run very slowly. IP Address: sovinity.com:25566
Once you are in the game, I suggest using the command "/top config" to change where WAILA shows up. Also, you can hold ~ (the button below escape) to mine lots of blocks at once, like a tree or a vein of ore.
